Visual C++

Включает Visual C++ Runtimes Components - с 2005 по 2022.

При свежей установке ОС Windows рекомендуется всегда устанавливать все различные среды выполнения C++.

Это помогает избежать проблем с программами, которые вызывают сообщения об ошибках типа:

  • side-by-side configuration is incorrect
  • MSVCRT.DLL
  • VCRUNTIME140_1.dll
  • MSVCP140.DLL

Все файлы имеют цифровую подпись Microsoft, что гарантирует их неизменность.
