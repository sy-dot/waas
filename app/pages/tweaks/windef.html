Отключение защитника Windows

Шаг 1

  1. Откройте диспетчер задач
  2. Перейдите в пункт «Автозагрузка»
  3. Выделите «Windows Security»
  4. Нажмите «Отключить»

Шаг 2

Если у вас домашняя редакция Windows - пропустите этот шаг.

Узнать какая у меня редакция Windows

  1. Откройте «Редактор групповой политики»
  2. «Конфигурация компьютера» >
  3. «Административные шаблоны» >
  4. «Компоненты Windows» >
  5. «Антивирусная программа Защитник Windows»
  6. Выставляем следующие параметры:
  • Выключить Microsoft Defender
  • Включено
  • Разрешить запуск службы защиты...
  • Отключено
  • Разрешить постоянную работу службы защиты...
  • Отключено
  • Проверять все загруженные файлы и вложения
  • Отключено
  1. Откройте подраздел «MAPS»
  2. Отключите все кроме «Отправлять образцы»
  • Отправлять образцы файлов
  • Включено
  • ...если требуется дальнейший анализ
  • Никогда не отправлять

Шаг 3

Скачайте файл твика.

Не открывайте этот файл НЕ в безопасном режиме!

Шаг 4

Перезагрузитесь в безопасном режиме.

Нажмите сюда для перезагрузки.

Шаг 5

В безопасном режиме

Примените твик (файл на рабочем столе).

После применения ваш ПК будет перезагружен в нормальное состояние.
